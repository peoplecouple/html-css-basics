box의 기본 성질 (display:block)

  • 투명하다.
  • width : auto / height : auto
  • Y축으로 쌓인다.
  • block 옆에 block이 오지 못하게 한다.
  1. 태그는 크게 inline / block 으로 나뉜다.
  2. 글자 그림 등 내용에 관여하는 태그 span strong em i 인라인 속성을 가지고 나머지는 block 속성을 가진다.
  3. block 속성은 width height padding margin 값을 갖는다. inline속성은 좌우 margin padding만 가진다.
  4. inline 상하 padding의 경우 background를 포함하여 padding이 적용된 것 처럼 보인다.
